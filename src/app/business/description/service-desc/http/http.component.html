HttpService

【说明】HttpService用于处理http请求。

HttpService提供接口方法包括：get()、post()、postFormData()

get()

用于提交get请求。方法入参如下：


get(
    url: string,
    paramMap: any = null,
    success: Function = (data) => { },
    error: Function = (msg) => { },
    extraConfig: any = null
): Subscription
返回值为请求的Subscription对象，可通过unsubscribe()方法取消订阅，终止请求回调函数。

post()

用于提交post请求，向指定资源提交数据进行处理请求（例如提交表单或者上传文件），数据被包含在请求体中。方法入参如下：


post(
    url: string,
    paramMap: any = null,
    success: Function = (data) => { },
    error: Function = (msg) => { },
    extraConfig: any = null
): Subscription
返回值为请求的Subscription对象，可通过unsubscribe()方法取消订阅，终止请求回调函数。

postFormData()

用于表单验证请求提交。方法入参如下：


postFormData(
    url: string,
    paramMap: any = null,
    success: Function = (data) => { },
    error: Function = (msg) => { },
    extraConfig: any = null
): Subscription
返回值为请求的Subscription对象，可通过unsubscribe()方法取消订阅，终止请求回调函数。
