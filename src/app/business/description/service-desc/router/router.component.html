RouterService

【说明】RouterService用于控制页面路由，实现页面跳转。

RouterService提供接口方法包括：url、gotoPage()、goBack()、getParamsCallBack()、getParamsSnapshot()
详情如下：

url

用于提交get请求。方法入参如下：


get url(): string
返回值为当前路由地址。

gotoPage()

用于路由跳转。方法入参如下：


gotoPage(
  url: string,
  params: Object = { }
): void
返回值为空。

goBack()

用于调用页面后退。方法入参如下：


goBack(): void
返回值为空。

getParamsCallBack()

用于通过回调获取当前路由参数。方法入参如下：


getParamsCallBack(
   func: Function = () => { }
): void
返回值为空。

getParamsSnapshot()

获取当前路由参数快照。方法入参如下：


getParamsSnapshot(): any
返回值为获得的当前路由参数对象。
