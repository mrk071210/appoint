【说明】 公共服务包括框架基础的底层服务。
【用法】 底层服务在app.module.ts中进行依赖注入，使用时如下使用：
【用法】最新版框架（2017年10月16日更新）已经将其加入到入口模块中，不再需要在app.module.ts进行额外的依赖，只需在业务代码中使用的地方导入即可。

import { HttpService, ... } from 'portalface/services'; //注：引用portalface库中的services文件夹下的服务
【特别说明】 Service服务需要在constructor中引用，Util公共方法则可直接通过类名进行方法调用。 

export class XXXX {

  constructor(private httpService: HttpService){
    }
  this.httpService.XX; // 调用时使用的是constructor中的实体对象httpService，而非HttpService类。
}
