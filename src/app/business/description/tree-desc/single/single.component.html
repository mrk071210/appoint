单选节点

该组件的树节点可以被单个选中。

Selected Node: {{selectedFile ? selectedFile.label : 'none'}}
HTML代码示例

<p-tree [value]="singleTree" selectionMode="single" [(selection)]="selectedFile" (onNodeSelect)="nodeSelect($event)"
(onNodeUnselect)="nodeUnselect($event)"></p-tree>
<div>Selected Node: {{selectedFile ? selectedFile.label : 'none'}}</div>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { TreeWidgetModule, TreeNode } from 'widgets'; // widgets 为根目录widgets文件夹

Step 2 HTML文件中使用基础树指令p-tree，属性value的值singleTree作为本示例的树节点，如下: 


<p-tree [value]="singleTree" selectionMode="single" [(selection)]="selectedFile" (onNodeSelect)="nodeSelect($event)"
(onNodeUnselect)="nodeUnselect($event)"></p-tree>
<div style="margin-top:8px">Selected Node:{{selectedFile ? selectedFile.label : 'none'}}</div>

Step 3 具体数据见TS示例文件。其中具体属性以及方法说明见简介部分。
