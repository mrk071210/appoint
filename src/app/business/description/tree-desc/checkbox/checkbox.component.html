初始化自动选中树节点示例代码

HTML代码示例

<p-tree [value]="checkboxTree" selectionMode="checkbox" [autoSelected]="true"></p-tree>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { TreeWidgetModule, TreeNode } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的FormWidgetsModule以及TreeNode

Step 2 HTML文件中使用基础树指令p-tree，属性value的值checkboxTree作为本示例的树节点，如下: 


<p-tree [value]="checkboxTree" selectionMode="checkbox" [(selection)]="selectedFiles"></p-tree>
<div>Selected Node:
  <span *ngFor="let file of selectedFiles">{{file.label}}</span>
</div>

Step 3 具体数据见TS示例文件。其中具体属性以及方法说明见简介部分。

初始化树节点自动选中使用说明

1、指定 p-tree 标签的 [autoSelected] 属性为"true"。

2、后台请求的树节点数据，选中节点的 checked 属性值为 true 。

3、使用 autoSelected 属性时，开发人员不必指定 p-tree 标签的 selection 属性的值，确保上一步 checked 属性赋值正确即可，其他的框架中已做处理。
