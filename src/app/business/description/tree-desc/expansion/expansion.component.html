伸缩

该组件的树节点可以全部展开和收起。
HTML代码示例

<div style="margin-bottom: 8px">
  <button pButton type="text" label="Expand all" (click)="expandAll()"></button>
  <button pButton type="text" label="Collapse all" (click)="collapseAll()"></button>
</div>
<p-tree #expandingTree [value]="expansionTree"></p-tree>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { TreeWidgetModule, TreeNode, Tree } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的TreeWidgetModule,TreeNode,Tree

Step 2 HTML文件中使用基础树指令p-tree，属性value的值expansionTree作为本示例的树节点，如下: 


<div style="margin-bottom: 8px">
  <button pButton type="text" label="Expand all" (click)="expandAll()"></button>
  <button pButton type="text" label="Collapse all" (click)="collapseAll()"></button>
</div>
<p-tree #expandingTree [value]="expansionTree"></p-tree>

Step 3 具体数据见TS示例文件。其中具体属性以及方法说明见简介部分。
