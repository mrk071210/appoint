基础树

该组件是最基础的用于分层显示数据的结构。
HTML代码示例

<p-tree [value]="basicTree"></p-tree>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { TreeWidgetModule, TreeNode } from 'widgets'; // widgets 为根目录widgets文件夹

Step 2 HTML文件中使用基础树指令p-tree，属性value的值basicTree作为本示例的树节点，如下: 


<p-tree [value]="basicTree"></p-tree>

Step 3 具体数据见TS示例文件。其中节点数据的属性说明见简介部分。
