下拉树

该组件是基于树组件和下拉组件封装的下拉树组件（非primeng原生）。
HTML代码示例

<p-droptree [value]="dropdownTree"></p-droptree>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { TreeWidgetModule, TreeNode } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的TreeWidgetModule,TreeNode

Step 2 HTML文件中使用基础树指令p-droptree，属性value的值dropdownTree作为本示例的树节点，如下: 


<p-droptree [value]="dropdownTree"></p-droptree>

Step 3 具体数据见TS示例文件。其中节点数据的属性说明见简介部分。
