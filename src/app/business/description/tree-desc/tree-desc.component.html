树

该组件用于显示分层数据。

import { TreeModule,TreeNode } from 'widgets'; // widgets 为根目录widgets文件夹

使用

树组件需要一个TreeNode对象数组作为节点内容。我们从TreeNode API开始了解树组件的使用详情。请注意，下面表格的所有属性都是可选的。

通过服务从json文件获取节点数据

大多数情况下，节点将从远程数据源加载，这里是一个从json文件中获取数据的NodeService示例。
这个.json文件由示例数据组成，在实际的应用程序中，应该是来自远程动态响应的。
使用该服务调用getfile()函数，并将内容分配给绑定到树的文件。



<p-tree [value]="files"></p-tree>

选择

树组件支持3种选择方法：单选、多选和复选框。选择类型是通过设置selectionMode属性设置，并提供单个TreeNode或TreeNode数组来绑定已选中的节点。



<p-tree [value]="files" selectionMode="single" [(selection)]="selectedFile"></p-tree>
在多选模式或复选框模式下，selection属性值应该是一个数组。
在多选模式下，可以使用metaKey或者根据默认情况下值为true的metaKeySelection属性单独切换项目。启用触摸功能的设备将会自动关闭metaKeySelection。



<p-tree [value]="files" selectionMode="multiple" [(selection)]="selectedFiles"></p-tree>
在复选框模式下，可以通过propapageSelectionDown和propagateSelectionUp属性设置是否需要上下传播选择状态。

<p-tree [value]="files" selectionMode="checkbox" [(selection)]="selectedFiles" [propagateSelectionUp]="false" [propagateSelectionDown]="false"></p-tree>
树组件提供onNodeSelect和onNodeUnselect两个方法作为选择状态的回调。

<p-tree [value]="files" selectionMode="single" [(selection)]="selectedFiles" (onNodeSelect)="nodeSelect($event)"></p-tree>
通过将节点的selectable属性设置为false，可以禁用特定节点的选择。

图标

树节点的图标是使用图标属性定义的，如果您需要一个扩展或折叠状态的图标，则使用扩展图标名和折叠图标名。

ContextMenu

通过将菜单实例绑定到树，树具有与ContextMenu的唯一集成。

<p-tree [value]="files" selectionMode="single" [(selection)]="selectedFile" [contextMenu]="cm"></p-tree>

<p-contextMenu #cm [model]="items"></p-contextMenu>
