ECharts介绍

【解释】 ECharts组件基于ECharts 3.4.x，是百度出品的基于HTML5的图表库，底层依赖轻量级的Canvas类库ZRender。
【说明】 用于在页面上显示图表。指令是kyeeEcharts。
【用法】 需要在使用ECharts介绍组件的模块引入ECharts介绍资源。 

import { EchartsWidgetModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的EchartsWidgetModule

绘制一个简单的图表

【说明】在绘图前我们需要为 ECharts 准备一个具备高宽的 DOM 容器。
HTML代码示例

<body>
    <!-- 为 ECharts 准备一个具备大小（宽高）的 DOM -->
    <div kyeeEcharts [options]="options" style="width: 600px;height:400px;"></div>
</body>
通过设置 options 属性生成一个简单的柱状图，下面是完整代码。
TS代码示例

export class EchartsDescComponent implements OnInit {
    options;
    constructor() { 
    }

    ngOnInit() {
        this.options = {
            title: {
                text: 'ECharts 入门示例'
            },
            tooltip: {},
            legend: {
                data:['销量']
            },
            xAxis: {
                data: ["衬衫","羊毛衫","雪纺衫","裤子","高跟鞋","袜子"]
            },
            yAxis: {},
            series: [{
                name: '销量',
                type: 'bar',
                data: [5, 20, 36, 10, 10, 20]
            }]
        };
    }
}
