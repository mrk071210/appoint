环形图

环形图是饼图的变体，空白中心允许包含整个数据的其他信息。

常规环形图

环形图（设置悬浮效果）
常规环形图HTML代码示例

<p-chart type="doughnut" [data]="doughnutData" > </p-chart>
环形图（设置悬浮效果）HTML代码示例

<p-chart type="doughnut" [data]="doughnutHoverData" > </p-chart>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { ChartsWidgetModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的ChartsWidgetModule

Step 2 HTML文件中使用图表指令p-chart， 设置类型 type 为doughnut如下: 


<p-chart type="doughnut" [data]="doughnutData" ></p-chart>
属性值说明
datadoughnutData图表数据

Step 3 TS文件中设置图表展示数据,详见组件源码。

数据集options列表

环形图允许为每个数据集指定多个属性,这些用于设置特定数据集的显示属性。

配置options列表

这些是Doughnut图表特有的定制选项。 这些选项与全局图配置选项合并，并形成图表的选项。
