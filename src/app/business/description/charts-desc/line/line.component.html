折线图

折线图或线图是一种图表，它将信息显示为通过直线段连接的称为“标记”的一系列数据点。
HTML代码示例

<p-chart type="line" [data]="lineData" ></p-chart>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { ChartsWidgetsModule } from 'widgets'; // widgets 为根目录widgets文件夹

Step 2 HTML文件中使用图表指令p-chart， 设置类型 type 为line如下: 


<p-chart type="line" [data]="lineData" ></p-chart>
属性值说明
datalineData图表数据

Step 3 TS文件中设置图表展示数据,详见组件源码。

图表参数options列表

折线图允许为每个数据集指定多个属性, 所有点*属性都可以指定为数组。 如果这些值设置为数组值，则第一个值适用于第一个点，第二个值应用于第二个点，依此类推。
