柱状图

柱状图或条形图是一种将数据进行分组并以长条形进行展示的图，这些长条的长度与需要展示的数据值成比例。

横向柱状图

常规柱状图（隐藏图例）
常规柱状图HTML代码示例

<p-chart type="bar" [data]="barData" [options]="barOptions" > </p-chart>
横向柱状图HTML代码示例

<p-chart type="horizontalBar" [data]="barData" > </p-chart>
常规柱状图（隐藏图例）HTML代码示例

<p-chart type="bar" [data]="hideLegendBarData" [options]="hideLegendOptions" > </p-chart>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { ChartsWidgetModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的ChartsWidgetModule

Step 2 HTML文件中使用图表指令p-chart，如下:

【常规柱状图】设置类型 type 为bar 


<p-chart type="bar" [data]="barData" ></p-chart>

【横向柱状图】设置类型 type 为horizontalBar


<p-chart type="horizontalBar" [data]="barData" ></p-chart>
属性值说明
databarData图表数据

Step 3 TS文件中设置图表展示数据,详见组件源码。

图表参数options列表

条形图允许为每个数据集指定多个属性,一些属性可以指定为数组。如果这些值设置为数组值，则第一个值适用于第一个条，第二个值应用于第二个条，等等。
