极地区域图

极地区域图与饼图类似，但每个段具有相同的角度 - 段的半径根据值而有所不同。
HTML代码示例

<p-chart type="polarArea" [data]="polarAreaData" > </p-chart>
TS代码示例

Step 1 TS文件中引入组件所需资源，如下: 


import { ChartsWidgetsModule } from 'widgets'; // widgets 为根目录widgets文件夹

Step 2 HTML文件中使用图表指令p-chart， 设置类型 type 为polarArea如下: 


<p-chart type="polarArea" [data]="polarAreaData" ></p-chart>
属性值说明
datapolarAreaData图表数据

Step 3 TS文件中设置图表展示数据,详见组件源码。
