弹出消息

弹出消息用于在叠加层中显示消息。

 

<p-growl [(value)]="msgs"></p-growl>

<div>
    <button type="button" pButton (click)="showSuccess()" label="Success" class="ui-button-success"></button>
    <button type="button" pButton (click)="showInfo()" label="Info" class="ui-button-info"></button>
    <button type="button" pButton (click)="showWarn()" label="Warn" class="ui-button-warning"></button>
    <button type="button" pButton (click)="showError()" label="Error" class="ui-button-danger"></button>
    <button type="button" pButton (click)="showMultiple()" label="多条"></button>
    <button type="button" pButton (click)="clear()" icon="fa-close" style="float:right" label="清空"></button>
</div>

使用

Message接口指定单条消息，将id、severity、summary和detail作为属性定义。 在弹出消息上显示的消息是使用应该是一个Message实例数组的value属性定义的。

HTML代码示例

<p-growl [(value)]="msgs"></p-growl>
TS代码示例

import { Message } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的Message

export class MyModel {

    msgs: Message[] = [];

}

程度级别

以下是消息级别的可选值。

  • success
  • info
  • warn
  • error

显示消息

将消息添加到数组足以通过弹出消息显示它们。 通过弹出消息也可以通过从数组中删除消息的方法删除消息。

HTML代码示例

<p-growl [(value)]="msgs"></p-growl>
    
<button type="button" (click)="show()">Show</button>
<button type="button" (click)="clear()">Clear</button>
TS代码示例

show() {
    this.msgs.push({severity:'info', summary:'Info Message', detail:'PrimeNG rocks'});
}

clear() {
    this.msgs = [];
}
