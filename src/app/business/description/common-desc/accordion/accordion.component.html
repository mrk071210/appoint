TS代码示例

export class AccordionComponent {

    msgs: Message[];
    
    index: number = 0;

    onTabClose(event) {
        this.msgs = [];
        this.msgs.push({severity:'info', summary:'Tab Closed', detail: 'Index: ' + event.index});
    }
    
    onTabOpen(event) {
        this.msgs = [];
        this.msgs.push({severity:'info', summary:'Tab Expanded', detail: 'Index: ' + event.index});
    }
    
    openNext() {
        this.index = (this.index === 3) ? 0 : this.index + 1;
    }
    
    openPrev() {
        this.index = (this.index === 0) ? 3 : this.index - 1;
    }
}

使用

手风琴面板元素由一个或多个 p-accordionTab 元素组成。 标签的标题使用 header 属性定义。


<p-accordion>
    <p-accordionTab header="Header 1">
       Content 1
    </p-accordionTab>
    <p-accordionTab header="Header 2">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Header 3">
        Content 3    
    </p-accordionTab>
</p-accordion>

选择

默认展示内容使用支持单向或双向绑定的selected属性进行指定。


<p-accordion>
    <p-accordionTab header="Header 1" [selected]="true">
       Content 1
    </p-accordionTab>
    <p-accordionTab header="Header 2">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Header 3">
        Content 3    
    </p-accordionTab>
</p-accordion>

多标签展示

默认情况下，一次只能有一个选项卡处于活动状态，而multiple属性允许多个选项卡同时处于活动状态。


<p-accordion [multiple]="true">
    <p-accordionTab header="Header 1">
        Content 1
    </p-accordionTab>
    <p-accordionTab header="Header 2">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Header 3">
        Content 3    
    </p-accordionTab>
</p-accordion>

不可用

可以通过将disabled属性设置为true来禁用选项卡。


<p-accordion>
    <p-accordionTab header="Header 1" [disabled]="true">
       Content 1
    </p-accordionTab>
    <p-accordionTab header="Header 2">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Header 3">
        Content 3    
    </p-accordionTab>
</p-accordion>

程序化控制

可以使用activeIndex属性以编程方式控制选项卡，在单个模式下， 它应该是一个数字，并且在多个模式中是一个数字数组，用于定义活动选项卡的索引。

HTML代码示例

<button type="button" pButton icon="fa fa-chevron-up" (click)="openPrev()"></button>
<button type="button" pButton icon="fa fa-chevron-down" (click)="openNext()"></button>
        
<p-accordion [activeIndex]="index">
    <p-accordionTab header="Header 1">
       Content 1
    </p-accordionTab>
    <p-accordionTab header="Header 2">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Header 3">
        Content 3    
    </p-accordionTab>
</p-accordion>
TS代码示例

export class AccordionDemo {
    
    index: number = 0;

    openNext() {
        this.index = (this.index === 2) ? 0 : this.index + 1;
    }
    
    openPrev() {
        this.index = (this.index === 0) ? 2 : this.index - 1;
    }
}
