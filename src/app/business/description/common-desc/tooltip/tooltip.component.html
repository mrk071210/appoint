<input type="text" pInputText pTooltip="请输入工号" placeholder="焦点" tooltipEvent="focus" style="margin-left:.5em">

使用

Tooltip应用于具有pTooltip指令的元素，其中该指令的值定义要显示的文本。


<input type="text" pTooltip="请输入工号">

位置

定位tooltip有四种选择，默认值为“right”，替代方法为“top”，“bottom”，“left”。 使用tooltipPosition属性指定位置。 


<input type="text" pTooltip="请输入工号" tooltipPosition="top">

事件

默认情况下，Tooltip显示在其目标的悬停事件上，其他配置有当focus事件显示和blur事件隐藏。


<input type="text" pTooltip="请输入工号" tooltipPosition="top" tooltipEvent="focus">

延迟

默认情况下，工具提示立即显示或隐藏，但您可能会使用showDelay和hideDelay属性添加延迟，该属性以毫秒为单位接受数字值。


<input type="text" pTooltip="请输入工号" tooltipPosition="top" tooltipEvent="focus" showDelay="1000" hideDelay="500">

滚动

当放置在像overflow div这样的滚动容器中时，必须将tooltip追加到另一个具有相对位置的元素，而不是缺省的文档主体。


<div #container style="display:inline-block;position:relative">
    <input type="text" pInputText pTooltip="请输入工号" placeholder="Right" [appendTo]="container">
</div>
