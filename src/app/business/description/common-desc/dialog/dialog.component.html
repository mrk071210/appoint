使用

对话框用作容器，可见性由visible属性控制。

HTML代码示例

<p-dialog header="Title" [(visible)]="display">
    Content
</p-dialog>

<button type="text" (click)="showDialog()" pButton icon="fa-external-link-square" label="Show"></button>
TS代码示例

export class ModelComponent {

    display: boolean = false;

    showDialog() {
        this.display = true;
    }

}

默认情况下，隐藏对话框，并启用visible属性显示对话框。 由于visible支持双向绑定，使用关闭按钮隐藏对话框将可见状态更新为false。

宽度

当对话框的内容较多时，对话框宽度可能意外增长，在这种情况下，定义对话框或内容的固定宽度。


<p-dialog header="Title" [(visible)]="display" [width]="300">
    Content
</p-dialog>

页眉和页脚内容

页眉和页脚部分可用于包含自定义内容。 页眉和页脚组件应在您的组件的指令部分导入和定义，以使其正常工作。 


<p-dialog [(visible)]="display">
    <p-header>
        Header content here
    </p-header>
    Content
    <p-footer>
        //buttons
    </p-footer>
</p-dialog>

内部叠加

当对话框包括具有覆盖的其他组件（如下拉列表）时，覆盖部分由于溢出而不能超出对话框边界。 为了解决这个问题，您可以将叠加层附加到body或允许在对话框中溢出。 


<p-dialog>
    <p-dropdown appendTo="body"></p-dropdown>
</p-dialog>

or:
<p-dialog [contentStyle]="{'overflow':'visible'}">
    <p-dropdown></p-dropdown>
</p-dialog>
