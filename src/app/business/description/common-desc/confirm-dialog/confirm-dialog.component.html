确认弹出框

确认弹出框是利用观测值显示确认窗口，很容易，可以通过在同一组件的多个行动共享服务支持。
HTML代码示例

<p-growl [value]="msgs"></p-growl>      
<p-confirmDialog></p-confirmDialog>       
<button type="button" (click)="confirm1()" pButton icon="fa-check" label="Confirm"></button>       
<button type="button" (click)="confirm2()" pButton icon="fa-trash" label="Delete"></button>
TS代码示例

使用

验证对话框是使用p-confirmDialog标签定义的，并需要一个验证服务实例来通过调用确认方法来显示它。

HTML代码示例

<p-confirmDialog header="Confirmation" icon="fa fa-question-circle" width="425"></p-confirmDialog>       
<button type="button" (click)="confirm()" pButton icon="fa-check" label="Confirm"></button>
TS代码示例

确认方法接受一个确认实例，用于自定义对话UI，以及接受和拒绝操作。

对话框的属性以两种方式定义，消息、图标和标题属性可以使用确认方法定义，也可以声明在p -confirm mdialog的ng - template上。 如果这些值不太可能改变，那么声明式方法将是有用的，但是在ng模板中定义的属性仍然可以通过确认方法调用。 此外，在footer部分的按钮可以通过传递您自己的UI来定制，重要的是要使确认工作与自定义UI一起定义一个本地的ng模板变量， 并为您自己的按钮分配accept()- reject()方法。

HTML代码示例

<p-confirmDialog header="Confirmation" icon="fa fa-question-circle" width="425" #cd>
    <p-footer>
        <button type="button" pButton icon="fa-close" label="No" (click)="cd.reject()"></button>
        <button type="button" pButton icon="fa-check" label="Yes" (click)="cd.accept()"></button>
    </p-footer>
</p-confirmDialog>
