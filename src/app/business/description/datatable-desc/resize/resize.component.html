调整表格列宽

【简介】通过设置 resizableColumns 属性为‘true’，可以实现用户拖拽改变列宽的效果。

【说明】表格提供了onColumnResize回调方法，将宽度发生变化的列的header当做参数传递给该函数。

HTML代码示例

<h3 class="first">Fit Mode</h3>
<p-dataTable [value]="cars" resizableColumns="true">
    <p-column field="vin" header="Vin"></p-column>
    <p-column field="year" header="Year"></p-column>
    <p-column field="brand" header="Brand"></p-column>
    <p-column field="color" header="Color"></p-column>
</p-dataTable>

<h3>Expand Mode</h3>
<p-dataTable [value]="cars" resizableColumns="true" columnResizeMode="expand">
    <p-column field="vin" header="Vin"></p-column>
    <p-column field="year" header="Year"></p-column>
    <p-column field="brand" header="Brand"></p-column>
    <p-column field="color" header="Color"></p-column>
</p-dataTable>
TS代码示例

export class DataTableColResizeDemo implements OnInit {

    cars: Car[];

    constructor(private carService: CarService) { }

    ngOnInit() {
        this.carService.getCarsSmall().then(cars => this.cars = cars);
    }
}

两种不同的拖拽模式

属性 columnResizeMode 有两种可选值，分别是‘fit’和‘expand’，代表着两种不同的效果。 默认值为‘fit’。

fit模式

columnResizeMode 取值为 ‘fit’时，改变某列的宽度时，表格的整体宽度不会随着该列的改变而改变。相反，相邻列会自动改变宽度来适应。

expand模式

columnResizeMode 取值为 ‘expand’时，改变某列的宽度时，表格的整体宽度也会随着该列的改变而改变。其他列的宽度不会发生改变。
