数据分页

【简介】讲表格的 paginator 属性为‘true’，可以显示表格底部的分页区。

【说明】rows 属性表示每页可以展示的数据条数。

【说明】pageLinks 属性表示底部可见页数。

【说明】rowsPerPageOptions 属性的值为一个整数集合，用来切换每页可以展示的数据条数。

List of Cars
HTML代码示例

<p-dataTable [value]="cars" [rows]="10" [paginator]="true" [pageLinks]="3" [rowsPerPageOptions]="[5,10,20]">
    <p-header>List of Cars</p-header>
    <p-column field="vin" header="Vin"></p-column>
    <p-column field="year" header="Year"></p-column>
    <p-column field="brand" header="Brand"></p-column>
    <p-column field="color" header="Color"></p-column>
</p-dataTable>
TS代码示例

export class DataTablePaginatorDemo implements OnInit {

    cars: Car[];

    constructor(private carService: CarService) { }

    ngOnInit() {
        this.carService.getCarsMedium().then(cars => this.cars = cars);
    }
}
