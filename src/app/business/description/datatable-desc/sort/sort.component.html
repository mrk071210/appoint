数据排序

【简介】通常情况下，对需要排序的列设置其sortable属性为true即可实现常规的排序效果。

【简介】特殊情况下，需要实现多列组合排序的需求，可以通过设置sortMode属性为multiple，然后配合电脑系统的metakey按键实现多选效果。

【注意】metaKey是和系统有关的，在Windows键盘中是Windows键，在mac键盘中是Cmd键。

TS代码示例

export class DataTableSortDemo implements OnInit {

    cars1: Car[];
    
    cars2: Car[];

    constructor(private carService: CarService) { }

    ngOnInit() {
        this.carService.getCarsSmall().then(cars => this.cars1 = cars);
        this.carService.getCarsSmall().then(cars => this.cars2 = cars);
    }
}
