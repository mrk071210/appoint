资源文件引入


import { FormWidgetsModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的FormWidgetsModule

使用

InputTextarea应用于具有pInputTextarea指令的输入字段。

1. 默认使用

HTML代码示例

<textarea pInputTextarea></textarea>

2. 双向绑定

可以使用标准的ngModel指令绑定模型。

HTML代码示例

<textarea pInputTextarea [(ngModel)]="property"/></textarea>

3. 自动调整

在自动调整大小模式下，textarea会增加，而不是显示一个滚动条。 启用此功能时，需要定义行和列属性。

HTML代码示例

<textarea [rows]="5" [cols]="30" pInputTextarea [autoResize]="true"/></textarea>
