下拉列表多选框

MultiSelect用于从集合中选择多个项。

默认下拉列表（多选可检索）

Selected Cars: {{selectedCars}}

默认下拉列表（多选可检索）

HTML代码示例

<p-multiSelect [options]="cars" [(ngModel)]="selectedCars"></p-multiSelect>
<p>Selected Cars: {{selectedCars}}</p>
TS代码示例

资源文件引入


import { FormWidgetsModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的FormWidgetsModule

使用

1. 双向绑定

使用ngModel定义双向值绑定，多选择需要一组选项，每个选项应遵循示例中的cities定义标签值属性。

HTML代码示例

<p-multiSelect [options]="cities" [(ngModel)]="selectedCities"></p-multiSelect>
TS代码示例

2. 模型驱动形式

MultiSelect也可以用于模型驱动形式。

HTML代码示例

<p-multiSelect [options]="cities" formControlName="selectedCities"></p-multiSelect>

3. 模板形式

MultiSelect也可以使用模板自定义内容。以下为带图标文本列表的多选

HTML代码示例

<p-multiSelect [options]="cars" [(ngModel)]="selectedCars2" [panelStyle]="{minWidth:'12em'}">
    <ng-template let-car let-i="index" pTemplate="item">
        {{i}}
        <img src="assets/showcase/images/demo/car/{{car.label}}.png" style="width:24px;display:inline-block;vertical-align:middle"/>
        <div style="font-size:14px;float:right;margin-top:4px">{{car.label}}</div>
        </ng-template>
        </p-multiSelect>
<p>Selected Cars: {{selectedCars2}}</p>
