使用

1. 双向绑定

使用ngModel定义双向值绑定，selectbutton需要一个选项集合，其中每个选项应遵循如示例中的cities集合中定义标签值属性的规则。

HTML代码示例

<p-selectButton [options]="cities" [(ngModel)]="selectedCity"></p-selectButton>
TS代码示例

2. 多值

SelectButton允许默认情况下仅选择一个项目，而多个选项可以选择多个项目。 在多种情况下，model属性应该是不为null或未定义的数组实例。

HTML代码示例

<p-selectButton [options]="cities" [(ngModel)]="selectedCities"></p-selectButton>

3. 模型驱动形式

SelectButton也可以用于模型驱动形式。

HTML代码示例

<p-selectButton [options]="cities" formControlName="selectedCity"></p-selectButton>
