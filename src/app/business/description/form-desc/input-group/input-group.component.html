输入框组

文本，图标，按钮和其他内容可以通过包装插件并输入到.ui-inputgroup元素中来分组在输入旁边。 同一组中也可以使用多个插件。

默认输入

按钮-输入

多选框-输入

默认输入

HTML代码示例

<div class="ui-g">
  <div class="ui-inputgroup">
    <span class="ui-inputgroup-addon"><i class="fa fa-user"></i></span>
    <input type="text" pInputText placeholder="Username" [useClear]="true" class="kyee-input-clear">
  </div>
</div>
TS代码示例

按钮-输入

HTML代码示例

<div class="ui-g">
  <div class="ui-inputgroup">
    <input type="text" pInputText placeholder="Keyword" [useClear]="true">
    <button pButton type="button" icon="fa-search" class="ui-button-secondary"></button>
  </div>
</div>
TS代码示例

多选-输入

HTML代码示例

<div class="ui-g">
  <div class="ui-inputgroup">
      <span class="ui-inputgroup-addon"><p-checkbox></p-checkbox></span>
      <input type="text" pInputText placeholder="Username">         
  </div>
</div>
TS代码示例

【Step1】引入资源文件


import { FormWidgetsModule } from 'portalface/widgets'; //注：引用portalface库中的widgets模块下的FormWidgetsModule

【Step2】使用示例


<div class="ui-g">
  <div class="ui-inputgroup">
    <span class="ui-inputgroup-addon"><i class="fa fa-user"></i></span>
    <input type="text" pInputText placeholder="Username">
  </div>
</div>
