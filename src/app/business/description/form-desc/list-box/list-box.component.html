使用

1. 双向绑定

使用ng模型定义双向值绑定，列表框需要一组选项，其中每个选项应遵循定义标签值属性的“选定项”界面。

HTML代码示例

<p-listbox [options]="cities" [(ngModel)]="selectedCity"></p-listbox>
TS代码示例

2. 选择

列表框允许选择单个或多个项目，而复选框选项显示一个复选框来指示多个选择。在单一情况下，模型应该是单个对象引用，而在多种情况下应该是一个数组。可以使用metaKey或根据默认情况下为true的metaKeySelection属性值的值单独切换多个项目。启用触控功能的设备将会自动关闭MetKeySelection。

HTML代码示例

<p-listbox [options]="cities" [(ngModel)]="selectedCities"></p-listbox>
TS代码示例

3. 过滤

过滤允许使用头部的输入字段在列表中搜索项目。可以听过启用过滤器属性使用过滤功能。

HTML代码示例

<p-listbox [options]="cities" [(ngModel)]="selectedCities" filter="filter"></p-listbox>

4. 模型驱动形式

Listbox也可以在模型驱动的窗体中使用。

HTML代码示例

<p-listbox [options]="cities" formControlName="cities"></p-listbox>

5. 自定义内容

默认情况下，选项的标签用作项目的显示文本，对于自定义内容支持，可以定义ng模板，其中本地ng模板变量引用到选项集合中的选项。

HTML代码示例

<p-listbox [options]="cars" [(ngModel)]="selectedCar" style="width:190px">
    <ng-template let-car let-i="index" pTemplate="item">
        <div class="ui-helper-clearfix">
            <img src="assets/showcase/images/demo/car/{{car.label}}.png" style="display:inline-block;margin:5px 0 0 5px" width="48">
            <span style="font-size:20px;float:right;margin:15px 10px 0 0">{{i}} - {{car.value}}</span>
        </div>
    </ng-template>
</p-listbox>
