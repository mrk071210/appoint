使用

1. 双向绑定

双向值绑定使用标准ngModel指令引用的日期属性定义。

HTML代码示例

<p-calendar [(ngModel)]="value"></p-calendar>
TS代码示例

export class MyModel {
    value: Date;  
}

2. 模型驱动形式

日历也可以在模型驱动的形式中使用。

HTML代码示例

<p-calendar formControlName="date"></p-calendar>

3. inline属性

默认情况下，日历显示在弹出窗口中，需要为内联模式时，启用内联属性inline。

HTML代码示例

<p-calendar [(ngModel)]="value" [inline]="true"></p-calendar>

4. 选择模式

默认日历允许选择一个日期和多个日期，可以通过设置选择模式到多个。在这种情况下，日历更新了一个日期数组，在这个数组中，可以选择可选择数量的可选择的日期，可以使用maxDateCount属性来限制。第三种选择是允许选择基于两个值数组的范围模式，其中第一个值是开始日期，第二个值是结束日期。注意，当前不支持在多个和范围模式中使用时间。

5. dateFormat属性

默认日期格式为mm / dd / yy，可以使用dateFormat属性设置日期自定义格式。

HTML代码示例

<p-calendar [(ngModel)]="dateValue" dateFormat="dd.mm.yy"></p-calendar>

6. time属性

TimePicker启用了showTime属性，24（默认）或12小时模式是使用hourFormat选项配置。

HTML代码示例

<p-calendar [(ngModel)]="value" showTime="showTime" hourFormat="12"></p-calendar>
<p-calendar [(ngModel)]="value" showTime="showTime" hourFormat="24"></p-calendar>

7. 日期限制

通过设置readonlyInput属性为true禁用手动输入日期，限制可选日期使用minDate和maxDate选项。

HTML代码示例

<p-calendar [(ngModel)]="dateValue" [minDate]="minDateValue" [maxDate]="maxDateValue" readonlyInput="readonlyInput">></p-calendar>

8. 禁用特定日期

要禁用特定日期或日期，请将readonlyInput设置为true并使用disabledDates和/或disabledDays选项限制可选日期。

HTML代码示例

<p-calendar [(ngModel)]="dateValue" [disabledDates]="invalidDates" [disabledDays]="[0,6}" readonlyInput="readonlyInput">></p-calendar>

9. 日期本地化

框架目前提供汉化的功能，可以将 showLanguage 属性设置为“ch”显示成中文，或者在业务代码中将语言对象绑定到语言设置属性来定义不同语言和格式的本地化。（默认为英文，如需汉化推荐使用第一种方法，个别日历组件若需要语言自定义，可以考虑使用第二种方案）。

HTML代码示例(方案1)

<p-calendar [(ngModel)]="dateValue" showLanguage="ch"></p-calendar>
HTML代码示例(方案2)

<p-calendar [(ngModel)]="dateValue" [locale]="en"></p-calendar>
TS代码示例(方案2)

10. 自定义内容

Calendar UI使用p-header 和p-footer组件接受定制内容。

HTML代码示例

<p-calendar [(ngModel)]="dateValue">
    <p-header>Header</p-header>
    <p-footer>Footer</p-footer>
</p-calendar>

此外，可以使用带有pTemplate指令的ng模板进行模板化，其值为“date”。这是突出特定日期的方便特性。请注意， 传递给模板的隐式变量不是一个日期实例，而是一个表示日期的元数据对象，它包含“day”、“month”和“year”属性。下面的例子改变了每个月10到21日的背景颜色。

HTML代码示例

<p-calendar [(ngModel)]="date10">
    <ng-template pTemplate="date" let-date>
        <span [ngStyle]="{backgroundColor: (date.day < 21 && date.day > 10) ? '#7cc67c' : 'inherit'}" style="border-radius:50%">{{date.day}}</span>
    </ng-template>
</p-calendar>
